Support

Frequently asked questions

Quick answers for event access, capacity, and platform behavior.

Can I get a refund? +

Refund eligibility depends on the event. Some events may allow full or partial refunds before a cutoff date, while others may be non-refundable. If an event is canceled, attendees will be notified and refunds will be handled accordingly.

What happens if an event reaches capacity? +

Once an event reaches maximum capacity, additional going RSVPs are no longer accepted. Guests may still be able to mark themselves as interested if that option remains available.

How does the platform handle duplicate RSVPs? +

The backend validates each submission and prevents duplicate RSVP entries for the same attendee when matching rules are triggered.

Which routes power this platform? +

The platform uses /events and /rsvps, with optional event filtering through ?event_id=.
